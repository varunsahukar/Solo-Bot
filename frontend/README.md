# Frontend (Vite + React)

This folder contains a standalone Vite React frontend variant in the repository.

## Commands

```bash
npm install
npm run dev
npm run build
```

## API Expectations

- Backend default URL: `http://localhost:8000`
- API calls should be centralized through the local `src/api` layer.

## Notes

- This app is separate from the main `solo-tutor/frontend` workspace app.
- If you are working on the active product UI, use `solo-tutor/frontend` instead.
